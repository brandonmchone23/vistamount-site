The Problem

DIY cord-hiding usually ends in a code violation.

A lot of homeowners run a regular extension cord through the wall to hide their TV cord. That's a fire-code violation in every U.S. jurisdiction — standard power cords aren't rated for in-wall use. The right way is a recessed in-wall power relocation kit (Datacomm, PowerBridge) which takes the existing outlet's power and re-presents it behind the TV through code-rated cable. Signal cables (HDMI, optical) can run freely inside the wall once the power side is handled correctly.

Common Questions

FAQs

Is in-wall power code-compliant?
Only if you use a UL-listed in-wall power relocation kit. Standard extension cords run through walls violate the National Electrical Code. We use Datacomm or PowerBridge kits that are explicitly approved for in-wall residential use.
Can you do in-wall on brick or stone walls?
No — in-wall concealment requires a hollow cavity for the cables. Solid brick or stone walls need a different solution: paint-matched on-wall raceway, or routing power down to the firebox if it's a fireplace install.
What if there's insulation in the wall?
Insulation can make fishing harder but rarely impossible. We use fish tape and rod sets designed for insulated walls. If we hit a true blockage we can't get around, we'll tell you on-site and offer alternatives (raceway, different routing).
Will I need a drywall patch after?
Usually no. Standard in-wall concealment is two clean cuts — one for the receptacle box at the TV, one near the floor for the source — and the receptacle plates cover both. Only complex routing through fire-blocked or odd walls requires access cuts that need patching.
How long does the install take?
Add 45-90 minutes to your standard TV mount. So a typical drywall install with in-wall concealment runs 2-3 hours total.
